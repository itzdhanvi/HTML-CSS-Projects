Frequently Asked Questions

  • HTML (HyperText Markup Language) is used to structure web content with elements like headings, paragraphs, and images.
  • CSS (Cascading Style Sheets) is used to style HTML elements, including colors, layouts, fonts, and animations.

CSS can be applied in three ways:

  • Inline CSS → Added directly to an element using the style attribute.
  • Internal CSS → Defined inside a <style> tag within the <head> of an HTML file.
  • External CSS → Written in a separate .css file and linked to HTML using link.

  • id → A unique identifier for one specific element (#id-name {} in CSS).
  • class → A reusable identifier for multiple elements (.class-name {} in CSS).

  • Pseudo-classes (:) → Apply styles when an element is in a certain state.
  • Pseudo-elements (::) → Style a specific part of an element.
